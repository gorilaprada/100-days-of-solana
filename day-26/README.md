# Day 26: Explore the [Solana Explorer](https://explorer.solana.com/)

## Lessons Learned:
- When browsing in the Solana Explorer [Verified Program Page](https://explorer.solana.com/verified-programs) I found [this program address](https://explorer.solana.com/address/14XDwwZQABkP3A8AqymsThii5o3FLULxZqLCkGbX1hHp) which was named Devtools. It had a link to [this GitHub repo](https://github.com/igneous-labs/S/tree/658f728dd476ca5f99ba11810e8ad8621caca70b) a project called LayerZero. From their [Whitepaper](https://layerzero.network/publications/LayerZero_Whitepaper_V2.1.1.pdf) they claim to be the first omnichain messaging protocol.

- From what I understand they use the different blockchains ways to make programs (e.g Solana's Program accounts, Ethereum's smart contracts) to code endpoints that can send information and thus enhance blockchain interoperatibility. They are only the providers of the messasing protocol, think a data exchange protocol like TCP/IP.

- It is interesting to see the plethora of different ideas that are being developed.
