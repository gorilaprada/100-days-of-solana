# Day 5: Reading about the different sorts of wallets.

## Small reflection on three wallet types

1. CLI wallet: 
  - Ease of use : Very easy
  - Security: Low
  - Use case: Development test environment

2. Browser wallet:
  - Ease of use : Easy
  - Security: Medium
  - Use case: Personal low value transactions that need to be approved quickly. Unless, you pair it with a cold wallet to sign transactions.

3. Mobile wallet:
  - Ease of use : Easy
  - Security: Medium
  - Use case: Personal low value transactions that need to be approved quickly. Unless, you pair it with a cold wallet to sign transactions.
